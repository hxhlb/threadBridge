# threadBridge 0.1.0-rc.1

First public macOS RC for threadBridge.

Highlights:

- signed and notarized universal macOS desktop app bundle
- Telegram desktop runtime owner, local management surface, and workspace runtime control path included
- release runtime data defaults to `~/Library/Application Support/threadBridge`

Known limitations:

- Homebrew tap publication is not included in this RC
- release automation is currently local-operator driven
